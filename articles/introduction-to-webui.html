Introduction to WebUI

Learn about WebUI, a Swift-based framework for building fast and secure web applications with a SwiftUI-inspired syntax.

What is WebUI?

WebUI is a Swift-based web framework designed for building fast, secure, and performant web applications. Its declarative syntax, inspired by SwiftUI, allows developers to create HTML, CSS, and JavaScript with minimal code. WebUI prioritizes simplicity, making it ideal for static sites and dynamic applications alike.

Why Did I Build It?

Addressing Tooling Gaps

Existing frameworks like WebUI.Anchor offered some of what I needed, but often fell short in customization or ease of use. I built WebUI to streamline my workflow, leveraging Swift’s type safety and a SwiftUI-like syntax to create a framework tailored to my styling preferences and long-term maintenance needs.

Key Features

Declarative Syntax

WebUI’s declarative approach lets you define your UI hierarchically, mirroring SwiftUI. This makes it intuitive to structure complex layouts with minimal boilerplate.

Rich Component Library

The framework provides components like WebUI.Code, WebUI.Code, and WebUI.Code, all styled with Tailwind CSS via the WebUI.Code method for flexibility.

Efficient Static Builds

WebUI generates minified HTML for static sites, optimized for performance and SEO. The WebUI.Code struct manages routes and builds your site with ease.

Example Usage

Building a Simple Page

Below is an example of a basic WebUI page with a heading and an image:
import WebUI struct SimplePage: HTML { func render() -> String { Article { ArticleSection(title: "Hello, WebUI!") { Text { "Welcome to my first WebUI page." } Image( source: "/images/sample.jpg", description: "Sample image" ) } }.render() } }

Adding Metadata

WebUI simplifies SEO with the Metadata and Document structs:
let metadata = Metadata( title: "Simple Page", description: "A WebUI demo page", image: "/images/sample.jpg", type: .article ) let document = Document( path: "/simple-page", metadata: metadata ) { SimplePage() }
